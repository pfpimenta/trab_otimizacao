TRABALHO DE OTIMIZACAO COMBINATORIA

intruçoes de como executar o algoritmo:
    -se certificar que Python 2.7 está instalado no computador
    -chamar pela linha de comando 'python genOtimizado.py inputFile seed populationSize'
        onde 'inputFile' é o endereço para o arquivo de entrada, 
            'seed' é uma string que será utilizada para a geração de numeros aleatorios
            e 'populationSize' é um número inteiro que diz o tamanho das populações (metaparametro do algoritmo genetico)
